## 아이디어 배경

사주를 보는 사람들이 많은데, 커리어 관점에서 분석해주는 서비스는 없었다. 사주팔자의 오행 균형을 분석해서 어떤 직무/산업이 맞는지 알려주면 재밌지 않을까?

## 기획 과정

### 수익 모델
- 기본 분석: 무료
- 상세 리포트: 유료 (미구현)
- 어필리에이트: 관련 교육/강의 연결

### 의사결정
처음에는 타로 + 사주를 합치려 했는데 스코프가 너무 커져서 사주만으로 축소했다.

## 커밋 로그

### 2026-04-05: 아이디어 시작
사주 관련 유튜브를 보다가 "이걸 커리어에 적용하면?"이라는 생각이 들었다.

### 2026-04-10: 첫 프로토타입
Claude API로 사주 분석 프롬프트를 만들었는데, 결과가 너무 일반적이었다. 오행 데이터를 구조화해서 넣어야 했다.

### 2026-04-15: 방향 전환
만세력 API를 찾아서 연동. 생년월일시를 입력하면 사주팔자를 자동으로 계산하고, 이를 바탕으로 분석하도록 변경.

### 2026-04-20: MVP 완성
jeonseonggi.com으로 배포. 하루 100명 정도 방문.

## 회고

AI 프롬프트의 품질이 핵심이었다. 사주 데이터를 구조화해서 넣으니 결과가 확 좋아졌다. 무료 서비스로 돌리기엔 API 비용이 걱정되지만, 일단 MVP니까 괜찮다.
